# Java_AccountingSystem
# 💰 Java Expense Tracker

This is a CLI-based expense tracking system built with Java.

## 📌 Features

- Add income and expense records
- Categorize expenses (food, transport, fun)
- View all records
- Calculate total balance
- Category-based expense analysis
- Category ranking (sorted)
- Monthly expense tracking
- Data persistence (save & load from file)
- Input validation and error handling

## 🧠 Concepts Used

- Object-Oriented Programming (OOP)
- ArrayList (data storage)
- HashMap (data aggregation)
- File I/O (save/load data)
- LocalDate (date handling)
- Exception Handling (try-catch)
- CLI Menu System (nested menu)

## 📂 Project Structure

- `Main.java` → User interface & control flow  
- `AccountManager.java` → Business logic  
- `Transaction.java` → Data model  
- `data.txt` → Stored records
